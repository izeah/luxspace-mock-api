# Luxspace-Mock-API

For fake project Luxspace-React-For-Designer

### How to Install

`npm run build`

### Endpoint List

1. Get Categories

`{HOST}/api/v1/categories?page=1&page_size=4`

2. Get Products

`{HOST}/api/v1/products?page=1&page_size=10`

3. Get Product Detail

`{HOST}/api/v1/products/:id`

4. Meta Checkout

`{HOST}/api/v1/checkout/meta`
